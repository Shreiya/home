Wee Wee Piggy


Wee Wee Piggy is a game for children and adults, built on HTML5, CSS3, and vanilla JavaScript. Using the arrow keys, the player must navigate through the forest to get piggy home. Upon each step, the player is provided an amusing message.

Woofer Onesie


Woofer Onesie is an Adafruit Flora project that reads the pulse of a puppy with the Pulse Amped sensor, and displays the heart beat with a running LED light across the body via a Neopixel strip.

  • Helpers :
    Arduino Language
