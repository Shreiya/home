Wee Wee Piggy


Wee Wee Piggy is a game for children and adults, built on HTML5, CSS3, and vanilla JavaScript. Using the arrow keys, the player must navigate through the forest to get piggy home. Upon each step, the player is provided an amusing message.

Feed My Fishy


Feed My Fishy is an IoT project using the Particle Photon that allows a person to remotely access their fish feeder via a webpage, and never worry about a hungry fishy again.

Woofer Onesie


Woofer Onesie is an Adafruit Flora project that reads the pulse of a puppy with the Pulse Amped sensor, and displays the heart beat with a running LED light across the body via a Neopixel strip.

  • Check it out here!


  • Helpers :
    Arduino Language

Where's My Wash?


Where's my Wash is an IoT project using the Particle Internet Button, checks for inertia from the washing machine and notifies the user once their clothes are washed and clean.

React Man


React Man is a two-player game, written in react.js. It plays on the logic on BomberMan that has two players on a single game board, trying to eliminate the other by deploying bombs with time intervals.
